# Photo Slider

A minimalist, responsive, and draggable photo slider component. It enables users to browse through a collection of images with a smooth horizontal dragging interaction.

## Features

- **Draggable Interface**: Smooth horizontal dragging with inertia.
- **Minimalist Design**: Clean black and white aesthetic.
- **Interactive**: Hover effects reveal saturation and icons.
- **Responsive**: Adapts to different screen sizes.

## Usage

Simply import the `PhotoSlider` component.

```tsx

export function Page() {
  return (
    <PhotoSlider />
  );
}
```
