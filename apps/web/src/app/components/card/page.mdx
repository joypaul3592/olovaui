## Usage

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from "@/app/components/card/card";
import { Button } from "@/app/components/button/button";

export default function Example() {
  return (
    <Card className="max-w-sm">
      <CardHeader>
        <div>
          <CardTitle>Starter plan</CardTitle>
          <CardDescription>Best for growing teams.</CardDescription>
        </div>
      </CardHeader>
      <CardContent className="text-sm text-muted-foreground">
        12 projects, shared folders, and priority support.
      </CardContent>
      <CardFooter>
        <Button size="sm">Choose plan</Button>
      </CardFooter>
    </Card>
  );
}
```

## API Reference

The `Card` components render semantic elements with `div`, `h3`, and `p` tags.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Controls card spacing and typography. |
| `className` | `string` | `undefined` | Add custom classes to any card slot. |
| `children` | `ReactNode` | `undefined` | Content of the slot component. |
| `aria-label` | `string` | `undefined` | Accessible label for actions when needed. |
