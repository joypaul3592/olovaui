A premium, horizontal slider inspired by Raycast's interface. It features landscape cards, strict snap-to-item mechanics, and a polished dark aesthetic.

## Features

- **Strict Snapping**: Drags snap precisely to the next or previous item.
- **Landscape Design**: Optimized for 16:10 or similar landscape content.
- **Focus Mechanics**: Active items are highlighted and scaled, while inactive ones recede.
- **Raycast Aesthetic**: Dark mode, glassmorphism, and subtle gradients.
