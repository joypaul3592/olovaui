## Features

- **Infinite Wrapping**: Draggable surface dynamically wraps items for an unbounded effect.
- **Momentum Physics**: Utilizes velocity and spring animations for realistic scrolling.
- **GPU Accelerated**: Smooth performance using Framer Motion transforms.
- **Responsive Widths**: Adapts card dimensions based on screen size.

## Usage

Simply import the `MomentumCarousel` component into your app.

```tsx
import MomentumCarousel from '@/components/ui/momentum-carousel';

export function Page() {
  return (
    <MomentumCarousel />
  );
}
```
